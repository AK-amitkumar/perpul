Extra Step Conditional

Customize your checkout flow.

If you want to use the extra information stage of checkout to make customers fill out a form before confirming their order, but you want to enable it only for a specific product category (Website Category).

Define a system parameter called bc_extra_step_conditional_categ which is the name of the Website Category you wish the extra info stage to be enabled for. You also need to enable the global switch of the website builder for the extra info stage.
